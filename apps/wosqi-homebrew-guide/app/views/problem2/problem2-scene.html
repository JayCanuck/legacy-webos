Problem: Driver installation failed

Possible Solutions:
  • If an older version of novacom is already installed, uninstall it.
     
  • Restart your computer, then try letting WOSQI install the drivers again
     
  • Try manually installing the driver. You can download it from http://bit.ly/wosqi-driver-win32 for 32-bit computers or http://bit.ly/wosqi-driver-win64 for 64-bit computers.http://bit.ly/wosqi-driver-machttp://bit.ly/wosqi-driver-linux32 for 32-bit computers or http://bit.ly/wosqi-driver-linux64 for 64-bit computers.
